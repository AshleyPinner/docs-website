## Fonts

This is the basic font information, in case you need to emulate it somewhere.

<Table>
  <thead>
    <tr>
      <th width={150}>
        **Style rules**
      </th>

      <th>
        **Values**
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        Default font
      </td>

      <td>
        The default font is **Gotham Book**. This is a licensed font. The fallback sequence is:

        * Gotham-Book
        * Gotham A
        * Gotham B
        * Helvetica Neue
        * Helvetica
        * Droid-Sans
        * Arial
        * sans-serif
      </td>
    </tr>

    <tr>
      <td>
        Code font
      </td>

      <td>
        The default `code` and `pre` font is **Source Code Pro**. This is loaded as a Google font. The fallback sequence is:

        * Source Code Pro
        * Monaco
        * Courier
        * monospace
      </td>
    </tr>

    <tr>
      <td>
        Font sizes
      </td>

      <td>
        Line height is 160% of font size.

        * Default: 16px
        * H1: Do not use in page content. (It is generated automatically.)
        * H2: 30px
        * H3: 24px (avoid using)
        * H4, H5, H6: Do not use in page content.

        Avoid using H3. Do not use H4, H5, or H6. If you have that many sub-levels, consider restructuring the content or breaking it out into multiple documents.
      </td>
    </tr>
  </tbody>
</Table>
