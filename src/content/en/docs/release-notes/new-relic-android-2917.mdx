### Notes

The New Relic app is even better with an improved home screen, displaying all your favorite services, hosts, mobile and browser apps, key transactions and more. This enhanced landing page also allows you to search across services, apps and hosts!

### New features

* New home screen with favorite APM, Mobile, and Browser apps, Infrastructure hosts, key transactions, and plugin instances.
* New search in Home screen, to quickly search across APM, Mobile, Browser apps, hosts and key transactions

### Improvements

* Using new endpoints for faster data retrieval for most entity data
