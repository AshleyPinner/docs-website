### Notes

New Relic Infrastructure Agent builds are available for multiple platforms. See [Update the Infrastructure](https://docs.newrelic.com/docs/infrastructure/new-relic-infrastructure/installation/update-infrastructure-agent) agent for how to download and update the appropriate version of the agent for your platform.

Note also that this is a Windows only release.

### Bug fixes

* High Windows I/O read/write operations numbers now reporting correct network information.
