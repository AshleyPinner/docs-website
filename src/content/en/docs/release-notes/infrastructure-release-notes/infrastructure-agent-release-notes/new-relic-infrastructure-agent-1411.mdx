### Notes

A new version of the agent has been released. Follow standard procedures to [update your Infrastructure agent](/docs/infrastructure/new-relic-infrastructure/installation/update-infrastructure-agent).

### Bugfixes

* Fix log obfuscation feature applying to integration received arguments.
  * Bug was introduced on agent v1.4.0 affecting some OHIs arguments.
  * Symptom was: OHI receives obfuscated PASSWORD env vars with value `HIDDEN`.
