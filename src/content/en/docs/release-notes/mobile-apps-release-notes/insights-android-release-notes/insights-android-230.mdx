### Notes

Google release an update to their support library that fixed a crashing bug causing the Insights for Android app to crash when rotating the device while viewing Data Apps. This release also updates the NewRelic agent to the 5.8.4 release.

### Fixes

* Bad URL causing metric charts to not retrieve data
* Crash on rotation while in data apps
* Crash when viewing details on large tables
