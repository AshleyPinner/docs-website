### Notes

Version 2.0 release adds new Data Apps support to the V1 release of Insights. Now users can view their favorite Data Apps on their android phone or tablet.

### New features

* Data Apps support
* Night mode

### Improvements

* Improved formatting on charts
* Improved error handling for incomplete data
* Improved performance

### Fixes

* Crashing on some heatmaps
* Incorrect time display on some charts
* Bad formatting on No Password view
