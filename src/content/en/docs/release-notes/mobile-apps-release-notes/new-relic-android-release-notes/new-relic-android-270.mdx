This release is the first release to support all the data charts in New Relic Infrastructure.

* Infrastructure Processes support
* Infrastructure Storage support
* Infrastructure Network support

### Improvements

* Dynamically loading navigation menu
* Infrastructure charts are now interactive

### Bug fixes
