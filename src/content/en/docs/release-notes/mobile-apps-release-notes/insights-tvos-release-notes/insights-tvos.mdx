### Notes

Initial release of [Insights for tvOS](/docs/mobile-apps/insights-app/features/insights-apple-tv-app).

### New features

* View dashboards.
* View data apps.
* Set up a slideshow to rotate through selected dashboards and data apps.
* Use dark or light modes.
* View widget details and info.
