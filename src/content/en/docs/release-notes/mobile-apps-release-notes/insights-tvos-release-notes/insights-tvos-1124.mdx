### New features

* Added New Relic One dashboard layout support
* Added app info screen with links to license and privacy info

### Improvements

* Added in support to remove dashboards by long press from slideshow screen
* Added in support to remove all slideshow dashboards from settings screen

### Bug fixes

* Fixed a crash when viewing attribute sheet widgets with no data
* Fixes incorrect refresh time for line chart widgets
