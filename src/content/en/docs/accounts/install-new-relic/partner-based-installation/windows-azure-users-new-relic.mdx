[Windows Azure](https://www.windowsazure.com/ "Link opens in a new window") is a Platform as a Service (PaaS) solution capable of hosting web applications using various languages, including .NET and Node.js.

For more information about using the New Relic .NET agent, see:

* [Installing the .NET agent on Azure](/docs/dotnet/installing-the-net-agent-on-azure)
* [Azure Preview Portal](/docs/agents/net-agent/azure-installation/azure-preview-portal)
* [Azure Web Sites](/docs/dotnet/azure-web-sites)
* [Azure Cloud Services](/docs/dotnet/azure-cloud-services)

For more information about installing the New Relic Node.js agent, see [Node.js agent on Microsoft Azure](/docs/nodejs/nodejs-agent-on-microsoft-azure).
