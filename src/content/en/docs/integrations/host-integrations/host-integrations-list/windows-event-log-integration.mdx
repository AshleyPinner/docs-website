The Windows event log integration monitors and reports [Windows event log](https://msdn.microsoft.com/en-us/library/windows/desktop/aa385780(v=vs.85).aspx) entries to New Relic.

To get started, see the [Windows Event Log On-Host Integration README](https://github.com/newrelic/nri-windows-event-logs) on GitHub.

<Callout variant="tip">
  This integration is released as [open source on Github](https://github.com/newrelic/nri-windows-event-logs). A change log is also available there for the latest updates. Support for this integration is available with an [Expert Services subscription](https://www.newrelic.com/expertservices).
</Callout>
