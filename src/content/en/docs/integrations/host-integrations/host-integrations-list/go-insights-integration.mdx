The Go Insights integration queries events from, and inserts to, New Relic.

To get started, see the [Go Insights On-Host Integration README](https://github.com/newrelic/go-insights) on GitHub.

<Callout variant="tip">
  This integration is released as [open source on Github](https://github.com/newrelic/go-insights). A change log is also available there for the latest updates. Support for this integration is available with an [Expert Services subscription](https://www.newrelic.com/expertservices).
</Callout>
