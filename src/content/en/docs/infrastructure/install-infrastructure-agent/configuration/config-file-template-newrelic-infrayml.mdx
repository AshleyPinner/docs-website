The infrastructure agent reads the `newrelic-infra.yml` file for its [configuration](/docs/infrastructure/install-configure-infrastructure/configuration/configure-infrastructure-agent). You can find the latest configuration template in the [infra-agent repository on GitHub](https://github.com/newrelic/infrastructure-agent/blob/master/assets/examples/infrastructure/newrelic-infra-template.yml.example).

<Callout variant="tip">
  We recommend [linting](http://www.yamllint.com/) the YAML configuration files before using them to avoid formatting issues.
</Callout>

For more information on each setting, see [Infrastructure configuration settings](/docs/infrastructure/install-configure-manage-infrastructure/configuration/infrastructure-configuration-settings).

<Callout variant="important">
  Always [restart the agent](/docs/infrastructure-start-stop-restart-check-agent-status) or your web server after changing settings.
</Callout>
