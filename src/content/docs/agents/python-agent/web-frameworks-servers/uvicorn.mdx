Read about the requirements and tips for integrating our Python agent with an app that uses the [ASGI](https://asgi.readthedocs.io/en/latest/) server [Uvicorn](https://www.uvicorn.org/). To return to the general install instructions, go to [Install the Python agent](/docs/agents/python-agent/installation-configuration/python-agent-installation).

## Automatic initialization with admin script

If you start your app with `python app.py` and use our Python agent **version 5.20.0.149** or higher, you can use the recommended [admin script integration method](/docs/agents/python-agent/installation-configuration/python-agent-installation#integration). For example:

```
NEW_RELIC_CONFIG_FILE=path/to/newrelic.ini newrelic-admin run-python uvicorn path_to_app
```

## Python agent API

You can use the [Python agent API](/docs/agents/python-agent/python-agent-api) to monitor any ASGI server or framework. To do this, mark the [ASGI application entry point](/docs/agents/python-agent/python-agent-api/asgiapplication-python-agent-api) and set the [transaction names](/docs/agents/python-agent/python-agent-api/settransactionname-python-agent-api).

## Event loop diagnostic support

Our Python agent supports `asyncio` event loop diagnostics. For more information, see [Python event loop diagnostics](/docs/agents/python-agent/supported-features/python-event-loop-diagnostics).
