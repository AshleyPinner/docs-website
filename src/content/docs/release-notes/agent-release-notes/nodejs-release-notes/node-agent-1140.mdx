### Notes

* The node agent now instruments connections to Oracle Databases using the `oracle` driver. This patch was contributed by [ryanwilliamquinn](https://github.com/ryanwilliamquinn)
* Fixed an issue that would break kraken apps when the node agent was enabled. This patch was contributed by [Lenny Markus](https://github.com/lmarkus)
