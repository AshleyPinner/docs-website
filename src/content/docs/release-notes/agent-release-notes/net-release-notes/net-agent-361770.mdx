End of Life

<Callout variant="important">
  The end-of-life date for this agent version is July 29, 2019. To update to the latest agent version, see [Update the agent](/docs/agents/manage-apm-agents/installation/update-new-relic-agent). For more information, see [End-of-life policy](/docs/agents/manage-apm-agents/maintenance/new-relic-agent-plugin-end-life-policy).
</Callout>

### New Features

* **Flexible capturing of attributes**

  You can now fine tune the attributes the .NET agent sends to New Relic. For more information, see [.NET agent attributes](/docs/agents/net-agent/attributes/net-agent-attributes)

### Fixes

* Fixes a bug where external requests that occurred outside of a transaction were generating error traces when the response code from the external request was an error.
* Fixes a bug where environment wasn't showing up correctly in the New Relic UI.
