### Fixes

* Fixed a bug that caused a StackOverflow exception for drivers that execute a query on `Connection.getMetaData()`.

**Note:** This issue was introduced in Java agent [version 3.30.0](https://docs.newrelic.com/docs/release-notes/agent-release-notes/java-release-notes/java-agent-3300).
