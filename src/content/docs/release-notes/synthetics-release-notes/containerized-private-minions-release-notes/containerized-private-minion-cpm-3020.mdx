### New features

* Increased minion worker configuration limits
  * Increased the minion worker configuration limits to allow customers to specify a greater number of light and heavy workers per CPU
  * Up to 50 heavy workers
  * Up to 1250 light workers
  * See Heavy and Light worker limits [documentation here](https://docs.newrelic.com/docs/synthetics/synthetic-monitoring/private-locations/containerized-private-minion-cpm-configuration#environment-variables)
