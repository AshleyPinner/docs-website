### Notes

A new version of the agent has been released. Follow standard procedures to [update your Infrastructure agent](/docs/infrastructure/new-relic-infrastructure/installation/update-infrastructure-agent).

### Added

Added new NFSSample (event type is NFSSample) for gathering NFS client metrics. This is configurable by the following config option:

* `metrics_nfs_sample_rate` (default: 20)
* `detailed_nfs` (default: false)
