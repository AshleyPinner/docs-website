### Notes

A new version of the agent has been released. Follow standard procedures to [update your Infrastructure agent](/docs/infrastructure/new-relic-infrastructure/installation/update-infrastructure-agent).

### Improvements

* [Windows Docker](https://docs.newrelic.com/docs/infrastructure/new-relic-infrastructure/data-instrumentation/docker-instrumentation-infrastructure) support for Windows Server 2016 hosts. Only Windows server native containers are supported.

### Bug fixes

* Fix a bug that prevented protected processes from being recorded by the agent in Windows.
