### Notes

Release 3.0 adds some exciting new features to Insights for Android including Filtered Dashboards, Linked Dashboards, and Deep Linking.

### New features

* Filtered Dashboards
* Linked Dashboards
* Deep Links
* Sharing of Permalinks

### Improvements

* Less memory used for data apps
* Updating data apps

### Fixes

* Fixes NRQL bugs causing line charts to crash
