### Notes

This release contains an updated JavaScript library and should improve performance of the NRQL editor. We also updated our old font to a new Open Sans font that matches the New Relic website.

### New features

* Multi-facet chart support
* Multi-facet NRQL editor support

### Improvements

* Updated JS library

### Fixes

* Crash on dashboards with no charts or graphs
