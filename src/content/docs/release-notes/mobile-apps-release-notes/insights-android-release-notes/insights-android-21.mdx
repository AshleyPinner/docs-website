### Notes

Insights for Android now has Query Builder to help build / edit you NRQL queries to give you insight into your data while on your mobile device. You can also share the results of your custom NRQL query with any of your contacts as an image.

### New features

* Query Builder / Query Editor
* Sharing of charts created view Query Builder
* Filtering for Dashboards / Data Apps lists

### Improvements

* Horizontal bar chart scrollable content
* Improved shared content to limit file size and include all data
