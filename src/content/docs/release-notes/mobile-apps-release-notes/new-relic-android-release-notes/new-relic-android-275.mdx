### Notes

Release 2.7.5 introduces Fingerprint Authorization for refreshing session tokens on email / password accounts for devices that support the fingerprint API. We also updated the Login Screen to new clean design and improved link support for infrastructure links.

### New features

* Fingerprint Id
* Link context saving

### Improvements

* Updated Login screens
