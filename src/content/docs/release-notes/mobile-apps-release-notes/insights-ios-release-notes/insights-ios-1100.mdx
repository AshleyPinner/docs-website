### Notes

Now supporting the EU region!

### New features

* Added support for switching to accounts in the EU region
* Added support for typing FROM first in the NRQL bar, instead of SELECT

### Bug fixes

* Fixed a Billboard bug when the warning value was greater then the critical value
* Fixed a bug where non linked pie chart, bar chart, and faceted tables would not be filtered when a filter is selected
* Fixed a long standing bug where widgets would sometimes disappear from dashboards
* Fixed a few rare crashes
