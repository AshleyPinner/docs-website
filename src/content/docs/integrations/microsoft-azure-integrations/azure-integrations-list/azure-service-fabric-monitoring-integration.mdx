We offer a cloud integration for reporting your Azure Service Fabric data to our platform. Here we explain how to activate the integration and what data it collects.

## Activate the integration

To enable the integration follow standard procedures to [activate your Azure service](/docs/infrastructure/microsoft-azure-integrations/getting-started/activate-azure-integrations).

## Configuration and polling

You can change the polling frequency and filter data using [configuration options](/docs/integrations/new-relic-integrations/getting-started/configure-polling-frequency-data-collection-cloud-integrations).

Default [polling](/docs/infrastructure/microsoft-azure-integrations/getting-started/azure-integration-polling) information for the Azure Service Fabric integration:

* New Relic polling interval: 5 minutes

## Find and use data

To find your integration data, go to **[one.newrelic.com](https://one.newrelic.com/ "Link opens in a new window.") > Infrastructure > Azure** and select the integration.

Data is attached to the following [event types](/docs/insights/use-insights-ui/getting-started/introduction-new-relic-insights#event-type):
